## Features
- Parse Salesforce Apex with Tree-sitter.
- Local SQLite cache for instant query responses.
- Freemium model with Llama 3 (free) and GPT-4o/Claude (Pro).

## Requirements
- Node.js >= 18
- pnpm >= 8
- VS Code >= 1.85

## Setup
```bash
pnpm install
pnpm run build
# Press F5 in VS Code to debug
